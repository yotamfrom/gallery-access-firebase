# Gallery Access Application

![Deploy to Firebase Hosting on merge](https://github.com/yotamfrom/Gallery_access_1/actions/workflows/firebase-hosting-merge.yml/badge.svg)

This is a React-based web application for managing and viewing artwork collections.

## Tech Stack
- **Frontend**: Vite, React, TypeScript, Tailwind CSS, shadcn-ui
- **Backend / Integration**: Firebase Cloud Functions (connecting to FileMaker Data API)

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

## Deployment

The application is configured for deployment on Firebase Hosting.
```bash
npm run deploy
```
